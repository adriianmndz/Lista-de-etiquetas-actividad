< header class="ejemplo"> 		Esta etiqueta define la cabecera de un documento o sección de él. Este frecuenta poseer logos, menús de navegación, títulos, etc. Suele estar acompañada del atributo id y también de la etiqueta < h1>

id:Se utiliza para identificar un elemento de forma única en el documento.

class:Se utiliza para asignar una o más clases a un elemento, permitiendo aplicar estilos CSS específicos.
